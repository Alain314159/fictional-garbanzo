# NostrChat

Una aplicación de chat descentralizada, gratuita y privada usando el protocolo Nostr.

## 🚀 Empezar

### Requisitos

- Android Studio Hedgehog (2023.1.1) o superior
- JDK 17
- Android SDK 35
- Dispositivo o emulador con Android 8.0+ (API 26+)

### Pasos

1. **Clonar el repositorio**
```bash
git clone https://github.com/tu-usuario/nostrchat.git
cd nostrchat
```

2. **Abrir en Android Studio**
   - File → Open → Seleccionar la carpeta del proyecto
   - Esperar a que Gradle sincronice

3. **Ejecutar la app**
   - Click en el botón Run (▶️)
   - O usar: `./gradlew installDebug`

- **Lenguaje**: Kotlin 2.0+
- **UI**: Jetpack Compose + Material 3
- **Arquitectura**: MVVM + Clean Architecture
- **Nostr SDK**: Rhodium
- **DI**: Hilt
- **Base de Datos**: Room
