## 💕 Lo que hace ÚNICA a esta App

NostrChat no es solo otra app de chat. Es una **experiencia emocional** donde cada mensaje puede ser mágico:

- ✨ **Destellitos dorados** cuando escribes "te amo"
- 🐷🤗🐨 **Un botón de abrazo** que muestra a Cerdita y Koalita abrazándose
- 🎤 **Nubecitas bailando** mientras escuchas notas de voz
- 📅 **Recordatorios de aniversarios** con animaciones especiales
- 🖼️ **Fondos donde las mascotas te saludan** al abrir el chat
- 💕 **Corazones flotando** cuando escribes algo romántico

## 📱 Flujo Completo de Usuario

### 1️⃣ Onboarding - Creación de Perfil

```
┌─────────────────────────────────────┐
│     👋 Bienvenido a NostrChat       │
│         🐷 🌸 🐨                    │
│                                     │
│  Tu nombre de usuario:              │
│  [_________________________]        │
│                                     │
│  Elige tu avatar:                   │
│  ┌─────┐  ┌─────┐  ┌─────┐         │
│  │ 📷  │  │ 🐷  │  │ 🐨  │         │
│  │Foto │  │Cerdita│Koalita│        │
│  └─────┘  └─────┘  └─────┘         │
│                                     │
│     [ ✨ Crear Perfil ]             │
│                                     │
│  ⚠️ Claves generadas automáticamente│
└─────────────────────────────────────┘
```

**Lo que sucede:**
- Usuario pone nombre
- Elige avatar (Cerdita 🐷 / Koalita 🐨 / Foto)
- Click en "Crear Perfil"
- **Automáticamente:**
  - Se generan claves pública/privada
  - Private key → Android Keystore
  - Public key → Lista para copiar
  - Perfil se publica en Nostr

## ✨ Características Mágicas Detalladas

### 1. ✨ Efectos en Palabras Románticas

**Palabras que activan efectos:**

| Palabra | Efecto |
|---------|--------|
| "te amo" | 💛 Destellitos dorados + 💕 corazones |
| "te quiero" | 💗 Corazones rosados flotando |
| "eres hermoso/a" | ✨ Brillitos alrededor |
| "me encantas" | 🌟 Estrellitas doradas |
| "buenos días" | 🌅 Amanecer + 🐦 pajaritos |
| "buenas noches" | 🌙 Luna + ⭐ estrellitas |
| "feliz cumpleaños" | 🎉 Confeti + 🎈 globos |
| "te extraño" | 💭 Nubecita + 🌧️ lluvia suave |

**Configurable:**
- Activar/desactivar efectos
- Intensidad (suave/normal/intensa)
- Añadir palabras personalizadas

---

### 2. 🖼️ Fondos Personalizados

**Fondos predeterminados:**
- 🐷 Cerditas durmiendo (animado: respiración)
- 🐨 Koalitas en árboles (animado: hojas moviéndose)
- 🌸 Jardín de flores (animado: pétalos cayendo)
- ☁️ Cielo con nubes (animado: flotando)
- 🌙 Noche estrellada (animado: brillando)
- 🌈 Arcoíris (animado: nubes moviéndose)
- 💕 Corazones flotando (animado: subiendo)
- 🐷🐨 Mascotas juntas (animado: jugando)

**Fondos personalizados:**
- Subir imagen de galería
- Ajustar brillo/blur/escala
- Por contacto (cada chat tiene el suyo)

**Animación de bienvenida:**
- Al abrir chat: mascotas aparecen y saludan
- Duración: 2 segundos
- Se desvanece suavemente

---

### 3. 🎤 Notas de Voz Mágicas

**Animaciones durante reproducción:**

| Tipo | Descripción |
|------|-------------|
| ☁️ **Nubecitas** | Nubes flotando mientras se reproduce |
| 🌟 **Estrellitas** | Brillan al ritmo del audio |
| 🐷 **Cerditas** | Saltan siguiendo la onda |
| 🐨 **Koalitas** | Se balancean con el audio |
| 🎵 **Notas musicales** | Flotan hacia arriba |

**Visualizador mejorado:**
- Onda de audio con colores del tema
- Picos de audio con partículas
- Sincronizado con el volumen

### 5. 💕 Botón Mágico de Abrazo

**Ubicación:** En chat, junto al input de texto

**Al presionar:**
1. Pantalla se oscurece
2. 🐷 Cerdita y 🐨 Koalita aparecen
3. Se acercan lentamente
4. **Se abrazan tiernamente**
5. 💕 Corazones flotando alrededor
6. ✨ Destellitos dorados
7. Mensaje: "Te envío un abrazo 🐷🤗🐨"
8. Desvanecimiento gradual

**Tipos de abrazos:**
- 🐷🤗🐨 **Normal** (default)
- 💕 **Romántico** (muchos corazones)
- ⭐ **Amistad** (estrellitas brillantes)
- 🐷🐷🤗🐨🐨 **Grupal** (múltiples mascotas)

**Configurable:**
- Activar/desactivar botón
- Tipo predeterminado
- Sonido opcional
- Vibración suave
